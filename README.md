# DSCubed Kaggle Competition 2026

Website for DSCubed's flagship Kaggle data science competition, Sem 1 2026. The competition is open to all skill levels — whether you're a complete beginner or experienced in data science, everyone's welcome.

Built with Next.js and Tailwind CSS.

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
